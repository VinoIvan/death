# VCM Laser - Салон лазерной эпиляции

Веб-сайт для салона лазерной эпиляции с системой онлайн-бронирования, админ-панелью и личным кабинетом клиента.

## 🚀 Функционал

**Клиентская часть:**
- Просмотр услуг с фильтрацией и сортировкой
- Онлайн-запись (выбор услуги, даты и времени)
- Личный кабинет с историей посещений
- Отзывы с рейтингом
- Контакты и форма обратной связи

**Админ-панель:**
- Управление записями (CRUD)
- Управление услугами и расписанием
- Управление пользователями и отзывами
- Статистика и графики

## 🛠 Технологии

**Backend:**
- PHP 8.2 / Laravel 10
- MySQL
- Laravel Breeze (аутентификация)

**Frontend:**
- Bootstrap 5
- jQuery
- Font Awesome 6
- AOS (анимации)
- Chart.js (графики)
- FullCalendar (календарь)
