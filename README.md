## 🚀 Функционал

### Клиентская часть
- ✅ Просмотр услуг с фильтрацией и сортировкой
- ✅ Онлайн-запись (выбор услуги, даты и времени)
- ✅ Личный кабинет с историей посещений
- ✅ Отзывы с рейтингом
- ✅ Контакты и форма обратной связи
- ✅ FAQ с аккордеоном
- ✅ Адаптивный дизайн для всех устройств

### Админ-панель
- ✅ Управление записями (CRUD) с фильтрацией
- ✅ Управление услугами и расписанием
- ✅ Управление пользователями и отзывами
- ✅ Статистика и графики (Chart.js)
- ✅ Календарь записей (FullCalendar)

## 📦 Установка

### Требования
- PHP >= 8.2
- Composer
- MySQL >= 8.0
- Node.js & NPM (опционально)

### Пошаговая установка

```bash
# Клонировать репозиторий
git clone https://github.com/VinoIvan/death.git
cd death

# Установить зависимости PHP
composer install

# Установить зависимости JavaScript (опционально)
npm install

# Настроить окружение
cp .env.example .env
php artisan key:generate

# После запуска откройте:

Клиентская часть: http://127.0.0.1:8000
Админ-панель: http://127.0.0.1:8000/ad

# Полный дамп базы данных доступен в database/dumps/vcm_laser.sql

Версия MySQL: 8.0.43
Дата создания: 2026-03-05

# Восстановление из дампа
# Способ 1: Через командную строку

# Создать базу данных
mysql -u root -p -e "CREATE DATABASE IF NOT EXISTS vcm_laser CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;"
# Импортировать дамп
mysql -u root -p vcm_laser < database/dumps/vcm_laser.sql

# Способ 2: Через phpMyAdmin
Откройте phpMyAdmin (http://localhost/phpmyadmin)
Создайте БД vcm_laser (кодировка utf8mb4_unicode_ci)
Перейдите в созданную базу данных
Нажмите вкладку "Импорт"
Выберите файл database/dumps/vcm_laser.sql
Нажмите "Выполнить"
